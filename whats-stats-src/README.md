# WhatsStats (source)

Svelte 5 + Vite app. Production static files are written to **`../whats-stats/`** for GitHub Pages at `/whats-stats/`.

```bash
npm install
npm run dev    # open the /whats-stats/ URL Vite prints
npm run build
```

Commit both **`whats-stats-src/`** and the generated **`whats-stats/`** output when deploying.
